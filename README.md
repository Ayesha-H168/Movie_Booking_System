Movie Booking System – Full Stack Project (Using Django)

* Developed a full stack web application for booking movie tickets online.
* Implemented user authentication features such as signup, login, and logout.
* Created modules to display movies, theatres, show timings, and seat availability.
* Designed an interface for users to select seats and book tickets.
* Added movie review and rating functionality for users.
* Built the frontend using HTML and CSS for responsive UI.
* Managed backend logic and database operations using Python and Django.
* Used MYSQL database and Django ORM for data storage and management.
